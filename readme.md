# Pair Programming exercise

![image](https://github.com/hxtruong6/call-for-paper-webpage/assets/24609363/b3d63bcd-0dba-443f-b6fb-213c7e911579)

Write an function that converts time into the corresponding english words - eg. '8:30' -> 'half past eight'

## Examples

- '0:00' > 'midnight'
- '12:00' > 'midday'
- '2:00' > 'two o'clock'
- '2:03' > 'three past two'
- '2:11' > 'eleven past two'
- '2:15' > 'quarter past two'
- '2:30' > 'half past two'
- '2:33' > 'twenty seven to three'
- '2:40' > 'twenty to three'
- '2:45' > 'quarter to three'
- '2:55' > 'five to three'
